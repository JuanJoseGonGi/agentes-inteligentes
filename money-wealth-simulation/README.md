### Installing

A step by step series of examples that tell you how to get a development env running.

(Optional) Create a virtual environment

```shell
python3 -m venv venv
```

Install dependencies

```shell
pip install -r requirements.txt
```

## 🎈 Usage

Just run the server.py file

```shell
python3 server.py
```
